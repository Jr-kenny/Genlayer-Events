# Genlayer Events
## Concept
A dedicated webpage called Genlayer Events that showcases all community activities from the Genlayer Discord including quizzes, games, community meetings, and more.

## Structure
• 	Live & Upcoming Events Page (combined)
• 	Displays events currently happening or scheduled in the future.
• 	Past Events Page
• 	Archives completed events for reference and community history

## Event Data Source Options
. **Manual Input (Google Sheets)**  
   - Update from phone or anywhere.  
   - Webpage fetches data via Google Sheets API.
